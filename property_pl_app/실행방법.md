# Property P&L Builder — 실행 방법

---

## 사전 준비: Python 설치 확인

앱을 실행하려면 **Python 3.10 이상**이 필요합니다.

### Python이 이미 설치됐는지 확인하기

**Windows:** 시작 메뉴 → "명령 프롬프트" 검색 → 열기 → 아래 입력
```
python --version
```

**Mac:** Launchpad → "터미널" 검색 → 열기 → 아래 입력
```
python3 --version
```

→ `Python 3.10.x` 또는 그 이상이 표시되면 OK. 없으면 아래에서 설치.

### Python 설치 (없는 경우)
👉 https://www.python.org/downloads/ 접속 → 노란 "Download Python" 버튼 클릭 → 설치

> ⚠️ **Windows 설치 시 중요**: 설치 화면에서 **"Add Python to PATH"** 체크박스를 반드시 체크하세요!

---

## STEP 1. 압축 파일 풀기

다운받은 `property_pl_app.zip` 파일을 원하는 폴더에 압축 해제합니다.

- **Windows**: 파일 우클릭 → "모두 압축 풀기"
- **Mac**: 파일 더블클릭

압축을 풀면 `property_pl_app` 폴더가 생깁니다.

---

## STEP 2. 터미널(명령창) 열기

### Windows
1. `property_pl_app` 폴더를 열기
2. 폴더 주소창(파일 탐색기 상단)을 클릭
3. 주소 전체 지우고 `cmd` 입력 후 Enter
4. 검은 창(명령 프롬프트)이 해당 폴더에서 열립니다

### Mac
1. `property_pl_app` 폴더를 Finder에서 찾기
2. 폴더를 우클릭 → "폴더에서 터미널 열기"
   (없으면: 터미널 열기 → `cd ` 입력 후 폴더를 터미널 창으로 드래그 → Enter)

---

## STEP 3. 필요 패키지 설치 (처음 한 번만)

터미널에 아래를 입력하고 Enter. 설치에 1~2분 걸립니다.

**Windows:**
```
pip install -r requirements.txt
```

**Mac:**
```
pip3 install -r requirements.txt
```

> 설치 중 에러가 나면: `pip install streamlit openpyxl pdfplumber pandas` 를 대신 입력하세요.

---

## STEP 4. 앱 실행

**Windows:**
```
python -m streamlit run app.py
```

**Mac:**
```
python3 -m streamlit run app.py
```

잠시 후 터미널에 아래 메시지가 나타납니다:
```
  Local URL: http://localhost:8501
```

---

## STEP 5. 브라우저에서 접속

브라우저(Chrome, Safari, Edge 등)를 열고 주소창에 입력:
```
http://localhost:8501
```

🎉 앱이 열리면 완료입니다!

---

## 앱 종료 방법

터미널에서 **Ctrl + C** 를 누르면 앱이 종료됩니다.

---

## 다음에 다시 실행할 때

설치는 처음 한 번만 하면 됩니다. 다음부터는 STEP 4만 반복하면 됩니다.

**Windows:**
```
python -m streamlit run app.py
```

**Mac:**
```
python3 -m streamlit run app.py
```

---

## 자주 묻는 문제

| 증상 | 해결 방법 |
|---|---|
| `python`을 찾을 수 없다 | Python 설치 후 컴퓨터 재시작 |
| `pip`를 찾을 수 없다 (Windows) | `python -m pip install -r requirements.txt` 사용 |
| 포트 8501 이미 사용 중 | `streamlit run app.py --server.port 8502` |
| 브라우저가 자동으로 안 열림 | 주소창에 `http://localhost:8501` 직접 입력 |
| PDF 파싱이 안 됨 | Step 2에서 문서 타입을 수동으로 선택 후 재시도 |
